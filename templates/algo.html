  • Пользователь регистрируется на сайте;
  • Пользователь создает игры либо подключается к существующей;
  • Пользователь ждет присоединения других игроков;
  • Пользователь, создавший игру, запускает ее;
  • Нейронная сеть генерирует вопросы для каждого человека и для каждого раунда;
  • Каждый пользователь в каждом раунде получает по 2 вопроса;
  • Каждый пользователь отвечает на оба вопроса;
  • После этого происходит анонимное голосование, среди игроков, где выбирается наиболее понравившийся ответ;
  • По результатам голосования начисляются баллы;
  • В конце игры пользователь с наибольшим количеством баллов побеждает;
  • Игра заканчивается, комната удаляется.
